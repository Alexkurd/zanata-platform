Glossary Info Data Type

Information about a specific Glossary.

Namespace
http://zanata.org/namespace/api/
Properties
name data type type namespace min/max occurs description
srcLocale Glossary Locale Info element ns0 0/1 The glossary's source locale
transLocale Glossary Locale Info element ns0 0/unbounded The list of translated locale's available for the glossary

Example

<glossaryInfo xmlns="http://zanata.org/namespace/api/">
  <srcLocale>
    <locale alias="es" displayName="Spanish (Spain)" enabled="..." enabledByDefault="..." localeId="es-ES" nativeName="Español" pluralForms="nplurals=2; plural=(n != 1)" rtl="..."/>
    <numberOfTerms>2</numberOfTerms>
  </srcLocale>
  <transLocale>
    <locale alias="es" displayName="Spanish (Spain)" enabled="..." enabledByDefault="..." localeId="es-ES" nativeName="Español" pluralForms="nplurals=2; plural=(n != 1)" rtl="..."/>
    <numberOfTerms>2</numberOfTerms>
  </transLocale>
</glossaryInfo>

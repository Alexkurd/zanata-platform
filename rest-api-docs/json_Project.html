Project Data Type

Representation of the data within a Project resource

Properties
name data type constraints description
id string   The project identifier (slug)
defaultType string   The project type.
name string required, max size: 80, min size: 0
description string max size: 100, min size: 0
sourceViewURL string   The url to view the project's sources.
sourceCheckoutURL string   The url where to checkout the project's sources.
links array of object   Set of links managed by this project This field is ignored in PUT/POST operations
iterations array of Project Version   A list of versions (iterations) in the project
status Status   System state of the project

Example

{
  "id" : "my-project",
  "defaultType" : "Gettext",
  "name" : "My Project",
  "description" : "This is a sample project.",
  "sourceViewURL" : "http://source.view.com",
  "sourceCheckoutURL" : "http://source.checkout.com",
  "links" : [ { }, { } ],
  "iterations" : [ {
    "id" : "my-iteration",
    "links" : [ { }, { } ],
    "status" : "READONLY",
    "projectType" : "Podir"
  }, {
    "id" : "...",
    "links" : [ { }, { } ],
    "status" : "READONLY",
    "projectType" : "..."
  } ],
  "status" : "OBSOLETE"
}

User Account Resource

Represents user accounts in the system. username: User name that identifies an account

GET /accounts/u/{username}

Retrieves a user account.

Response Codes
code condition
200 response containing information for the user account
500 If there is an unexpected error in the server while performing this operation
Response Body
media type data type
application/vnd.zanata.account+json Account (JSON)
application/vnd.zanata.account+xml Account (XML)

Example

Request
GET /accounts/u/{username}
Accept: application/vnd.zanata.account+json
Response
HTTP/1.1 200 OK
Content-Type: application/vnd.zanata.account+json

PUT /accounts/u/{username}

Creates or updates a user account. If an account with the given user name already exists, said account will be overwritten with the provided data. Otherwise, a new account will be created.

Request Body
media type data type description
application/vnd.zanata.account+json Account (JSON) The account information to create/update.
application/vnd.zanata.account+xml Account (XML)
Response Codes
code condition
200 An existing account was modified
201 A new account was created
401 The user does not have the proper permissions to perform this operation
500 If there is an unexpected error in the server while performing this operation
Response Body
media type data type
application/json object (JSON)
application/xml anyType (XML)

Example

Request
PUT /accounts/u/{username}
Content-Type: application/vnd.zanata.account+json
Accept: application/json

  "email" : "email@example.com",

  "languages" : [ "es", "ja" ],
  "enabled" : true
}

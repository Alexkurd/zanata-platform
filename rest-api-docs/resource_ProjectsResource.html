Project List Resource

API endpoint to get a list of projects.

GET /projects

Retrieves a full list of projects in the system.

Response Codes
code condition
200 Response containing a full list of projects.
500 If there is an unexpected error in the server while performing this operation
Response Body
media type data type
application/json array of Project (JSON)
application/vnd.zanata.projects+json array of Project (JSON)
application/vnd.zanata.projects+xml list of Project (XML)
application/xml list of Project (XML)

Example

Request
GET /projects
Accept: application/json
Response
HTTP/1.1 200 OK
Content-Type: application/json
